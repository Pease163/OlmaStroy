# Админ-панель

## Вход

- **URL:** `/admin/login`
- **Логин (seed):** `admin`
- **Пароль (seed):** `admin123`

После успешной авторизации — редирект на `/admin/`.

Выход: `/admin/logout` → редирект на главную страницу.

## Dashboard

URL: `/admin/`

Отображает карточки со статистикой:

| Показатель | Описание |
|-----------|----------|
| Статьи | Общее количество записей в блоге |
| Вакансии | Общее количество вакансий |
| Проекты | Общее количество проектов |
| Непрочитанные заявки | Количество заявок с `is_read = False` |

## Управление блогом

**URL:** `/admin/blogpost/`

### Список статей

Таблица с колонками: ID, Заголовок, URL-адрес (slug), Опубликовано, Дата создания, Дата обновления.

- **Сортировка:** по дате создания (новые первыми)
- **Поиск:** по заголовку и содержимому
- **Фильтры:** по статусу публикации, по дате создания

### Создание статьи

**URL:** `/admin/blogpost/new/`

| Поле | Описание |
|------|----------|
| Заголовок | Название статьи |
| URL-адрес (slug) | Генерируется автоматически из заголовка при создании |
| Контент | WYSIWYG-редактор CKEditor (HTML) |
| Краткое описание | Текст для карточки в списке (excerpt) |
| URL изображения | Ссылка на изображение статьи |
| Опубликовано | Чекбокс — при снятии статья не отображается на сайте |

**Автогенерация slug:** При создании статьи slug генерируется автоматически из заголовка через транслитерацию. Пример: `«Завершение строительства КС»` → `zavershenie-stroitelstva-ks`.

### Редактирование статьи

**URL:** `/admin/blogpost/edit/<id>/`

Те же поля, что и при создании. Slug обновляется только если поле пустое.

## Управление вакансиями

**URL:** `/admin/vacancy/`

### Список вакансий

Таблица: ID, Должность, Местоположение, Тип занятости, Зарплата, Активна, Дата создания.

- **Поиск:** по должности и местоположению
- **Фильтры:** по статусу активности, типу занятости, местоположению

### Создание / редактирование вакансии

| Поле | Описание |
|------|----------|
| Должность | Название вакансии |
| Местоположение | Город или регион |
| Описание | WYSIWYG-редактор CKEditor (HTML) |
| Зарплата | Текстовое поле (например, `от 80 000 ₽`) |
| Требования | WYSIWYG-редактор CKEditor (HTML) |
| Тип занятости | Выпадающий список: Полная занятость, Частичная занятость, Вахта, Стажировка |
| Активна | Чекбокс — при снятии вакансия не отображается на сайте |

## Управление проектами

**URL:** `/admin/project/`

### Список проектов

Таблица: ID, Название, URL-адрес, Местоположение, Категория, Год, Порядок, Видимый.

- **Сортировка:** по полю «Порядок» (по умолчанию)
- **Поиск:** по названию, местоположению, категории
- **Фильтры:** по категории, году, видимости

### Создание / редактирование проекта

| Поле | Описание |
|------|----------|
| Название | Название проекта |
| URL-адрес (slug) | Генерируется автоматически из названия при создании |
| Местоположение | Регион/город |
| Описание | Краткое описание (текст) |
| Контент | Полное описание (textarea, HTML) |
| URL изображения | Ссылка на изображение проекта |
| Категория | Текстовое поле (Компрессорные станции, Магистральные газопроводы и т.д.) |
| Год | Год реализации (число) |
| Порядок | Числовой порядок сортировки (меньше = выше) |
| Видимый | Чекбокс — при снятии проект не отображается на сайте |

## Просмотр заявок

**URL:** `/admin/contact/`

Заявки доступны **только для просмотра** — создание, редактирование и удаление запрещены.

### Список заявок

Таблица: ID, Имя, Телефон, Email, Тема, Прочитано, Дата.

- **Сортировка:** по дате (новые первыми)
- **Фильтры:** по статусу «Прочитано», по дате
- **Детальный просмотр:** клик на запись → полная информация включая текст сообщения

### Bulk-действие

**«Отметить как прочитанные»** — выделите нужные заявки чекбоксами и выберите действие из выпадающего списка. Устанавливает `is_read = True` для выбранных записей.

## Управление пользователями

**URL:** `/admin/user/`

### Список пользователей

Таблица: ID, Логин, Email, Администратор, Дата создания.

- **Поиск:** по логину и email

### Создание / редактирование пользователя

| Поле | Описание |
|------|----------|
| Логин | Уникальное имя пользователя |
| Email | Уникальный email |
| Администратор | Чекбокс — даёт доступ к админ-панели |

> Поле `password_hash` исключено из форм. Для смены пароля используйте скрипт или shell.

## Техническая информация

- Админ-панель построена на **Flask-Admin** с темой **Bootstrap 4 (cosmo)**
- Кастомные стили: `app/static/css/admin-custom.css`
- Dashboard-шаблон: `app/templates/admin/index.html`
- Все views защищены через `AuthMixin` — только пользователи с `is_admin = True` получают доступ
