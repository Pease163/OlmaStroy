# ОлмаСтрой — Корпоративный сайт

> Корпоративный веб-сайт ООО «ОлмаСтрой» — генерального подрядчика строительства нефтегазовой инфраструктуры (с 2006 года).

## Назначение

Сайт решает следующие бизнес-задачи:

- **Представление компании** — информация о деятельности, проектах, географии работ
- **Публикация новостей** — блог с материалами о строительстве и технологиях
- **Привлечение кадров** — каталог вакансий с фильтрацией по локации и типу занятости
- **Сбор заявок** — контактная форма для потенциальных заказчиков и партнёров

**Целевая аудитория:** потенциальные заказчики, партнёры, соискатели.

## Быстрый старт

```bash
git clone <repository-url>
cd ОлмаСТРОЙ
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python seed.py
python run.py
```

Сайт будет доступен по адресу: `http://localhost:5001`

Админ-панель: `http://localhost:5001/admin` (логин: `admin`, пароль: `admin123`)
