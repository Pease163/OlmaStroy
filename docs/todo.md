# Известные ограничения и TODO

## Известные ограничения

### База данных
- **SQLite** — не подходит для production с высокой нагрузкой. Ограничения: одновременная запись, отсутствие сетевого доступа, ограниченная масштабируемость.

### Безопасность
- **Fallback secret key** — если переменная `SECRET_KEY` не задана в `.env`, используется захардкоженный `'fallback-secret-key'` в `config.py`. В production это критическая уязвимость.
- **Дефолтные credentials** — seed.py создаёт администратора `admin` / `admin123`. Необходимо сменить пароль после деплоя.

### Тестирование
- **Нет тестов** — отсутствуют unit-тесты, интеграционные тесты, e2e-тесты.

### CI/CD
- **Нет пайплайна** — отсутствует автоматическая сборка, тестирование и деплой.

### Контейнеризация
- **Нет Docker** — отсутствуют `Dockerfile` и `docker-compose.yml`.

### CDN-зависимости
- **Tailwind CSS** подключён через CDN (`cdn.tailwindcss.com`) — при недоступности CDN стили частично пропадут.
- **CKEditor** загружается через CDN — при недоступности редактор не будет работать.
- **Google Fonts** (Manrope, Oswald) — при недоступности CDN шрифты заменятся на системные.

### Производительность
- **logo-icon.svg** — размер файла 303KB, используется в прелоадере и хедере. Требует оптимизации (сжатие, удаление лишних метаданных).

### Git
- **Нет Git-flow** — отсутствует настроенная стратегия ветвления.

## Технический долг

### Tailwind CSS CDN → локальная сборка

Сейчас:
```html
<script src="https://cdn.tailwindcss.com"></script>
```

Рекомендуется:
```bash
npm install -D tailwindcss
npx tailwindcss -i ./src/input.css -o ./app/static/css/tailwind.css --minify
```

### CKEditor CDN → локальная версия

Сейчас CKEditor загружается с CDN. При установке `CKEDITOR_SERVE_LOCAL = True` Flask-CKEditor должен раздавать файлы локально, но рекомендуется проверить корректность работы.

### Оптимизация logo-icon.svg

Текущий размер: ~303KB. Рекомендуется:

```bash
npx svgo app/static/img/logo-icon.svg -o app/static/img/logo-icon.svg
```

Ожидаемый размер после оптимизации: < 50KB.

### Дублирование TRANSLIT_MAP

Таблица транслитерации `TRANSLIT_MAP` дублируется в двух файлах:
- `app/models/blog.py`
- `app/models/project.py`

Рекомендуется вынести в `app/utils.py` и переиспользовать.

### datetime.utcnow

Модели используют `datetime.utcnow` — в новых версиях Python (3.12+) рекомендуется `datetime.now(timezone.utc)`.
